### 🚀 Featured Projects

**[🩺 Incident Management AI](https://github.com/karthick-sanjay/incident-management-ai)**
AI-powered IT Incident Management platform — classifies incoming incidents by severity & category (XGBoost), retrieves likely root causes through a FAISS-based RAG pipeline, and uses Groq/Llama for reasoning, all served through a FastAPI backend with JWT auth and a full multi-page frontend.
`Python` `FastAPI` `Groq/Llama` `FAISS` `XGBoost` `SQLAlchemy`

**[⚙️ PLC Code Generator — IEC 61131-3](https://github.com/karthick-sanjay/ai-based-plc-code-narration-tool)**
Converts plain-English instructions into PLC programs across multiple IEC 61131-3 languages using the Groq AI API, built for industrial automation use cases, with simulation-based logic verification planned next.
`JavaScript` `Groq AI API`

**[🔄 AI-Based Code Translator](https://github.com/karthick-sanjay/ai-based-code-translator)**
Translates programs between languages without changing the core logic, using AI APIs for syntax-aware, automated conversion — built to simplify migration between tech stacks.
`JavaScript` `AI APIs`

**[📧 Dynamic Mail Automation Tool](https://github.com/karthick-sanjay/dynamic-mail-automation-tool)**
SMTP-based bulk mailing system with fully customizable templates (logos, colors, layouts, social links) for automated email campaigns.
`Python` `SMTP`

**[📝 Research Paper Generator Agent](https://github.com/karthick-sanjay/report-writer)**
A multi-agent pipeline — input processing → template analysis → literature review → citation management → section writing → document assembly — that turns raw project details into a fully formatted academic research paper (.docx) using an LLM.
`Python` `LLM APIs` `python-docx`

**[📄 Resume Analyzer](https://github.com/karthick-sanjay/resume-analyzer)**
A Python tool that reviews resumes and surfaces feedback to help job-seekers improve them.
`Python`

**Also explored:** a LoRa-based **Smart Disaster Shelter Management System** — sensor nodes plus a Wi-Fi gateway that deliver live shelter status (population, food availability, temperature, humidity) to affected people without internet connectivity during outages.
